# ElectroPremium - Premium Electronics E-commerce Template

ElectroPremium is a production-ready, Apple-inspired e-commerce template designed for high-end electronics stores. It features a minimalist aesthetic, dual-role dashboards, and a complete shopping flow.

## 🚀 Key Features
- **Apple-Style UI**: Minimalist, high whitespace, and premium monochromatic design.
- **Dual Dashboards**: Separate interfaces for Administrators and Users.
- **Responsive Design**: Mobile-first approach with optimized touch targets.
- **Dark Mode**: Supports system preference and manual toggling.
- **SEO Optimized**: Semantic HTML, JSON-LD structured data, and meta tags.
- **RTL Support**: Built-in support for right-to-left languages.

## 🛠️ Getting Started
1. Clone or download the repository.
2. Open `pages/index.html` in any modern web browser.
3. To customize colors, edit the `:root` variables in `assets/css/style.css`.
